# Figma를 활용한 웹 와이어프레임 작성 가이드

## 개요

이 문서는 **Figma**를 사용하여 웹 서비스의 와이어프레임을 그리는 방법을 단계별로 안내합니다. 와이어프레임은 UI/UX 설계의 초석으로, 실제 개발 전에 레이아웃과 구성 흐름을 명확히 하기 위한 시각 도구입니다.

---

## 준비사항

- [Figma 웹사이트](https://www.figma.com/) 가입 및 로그인
- 새 디자인 파일 생성 (`New design file`)
- Google Fonts 또는 Pretendard 폰트 설치 권장 (디자인 일관성 유지)

---

## 1. Figma 화면 구조 이해

| 영역 | 설명 |
|------|------|
| 왼쪽 패널 | 레이어 구조 / 페이지 목록 |
| 중앙 캔버스 | 디자인 작업 공간 |
| 오른쪽 속성창 | 선택된 요소의 세부 속성 (크기, 색상, 정렬 등) |
| 상단 툴바 | 도형, 텍스트, 프레임 생성 도구 |

---

## 2. 페이지 구조 설계 (기획 반영)

서비스 기획서를 참고하여 페이지 목록을 작성합니다.

예:
- 홈
- 채팅
- 아기 키우기
- 화상 통화
- 캘린더
- 4컷 만화
- 중재 역할
- 선물/데이트 추천

---

## 3. 프레임 만들기 (화면 단위 생성)

1. **단축키 `F` 또는 툴바에서 "Frame" 클릭**
2. 프레임 크기 설정
    - 웹 기준: `1440 x 1024` 또는 `1280 x 800`
3. 프레임 이름 설정
    - `홈`, `채팅`, `캘린더`

> Tip: 여러 프레임을 가로로 배치하면 페이지 흐름이 보이기 쉬워요.

---

## 4. 기본 컴포넌트 배치

| 요소 | 생성 방법 | 설명 |
|------|------------|------|
| 텍스트 | `T` → 캔버스 클릭 | 타이틀, 버튼, 설명 등 |
| 도형 | `R` (사각형), `O` (원) | 버튼, 이미지 영역, 카드 등 |
| 아이콘 | `Assets` 또는 `Plugins` 활용 | 예: Feather Icons, Material Symbols |
| 이미지 | `Shift + Ctrl/Cmd + K` | 아기 사진, 채팅 이미지 등 |

---

## 6. 반복 요소는 컴포넌트로 만들기

1. 버튼, 카드, 채팅버블 등 반복되는 요소는 **Components (`Ctrl/Cmd + Alt + K`)** 로 등록
2. 추후 디자인 변경 시 전체에 적용 가능

---

## 7. 와이어프레임 스타일 규칙

| 항목 | 스타일 |
|------|--------|
| 폰트 | Pretendard / Noto Sans KR |
| 색상 | 회색 계열 (#333, #666, #ccc 등) |
| 테두리 | 1px, 회색, 둥근 모서리 |
| 배경 | 흰색 또는 연한 회색 |
| 정렬 | Auto Layout 활용 권장 (`Shift + A`) |

---

## 8. 흐름 연결 (Prototype 기능)

1. `Prototype` 탭 클릭
2. 버튼 → 다음 프레임 연결 (예: 채팅 버튼 → 채팅 프레임)
3. `Present`로 실제 전환 흐름 확인

---

## 유용한 플러그인 추천

| 이름 | 기능 |
|------|------|
| Unsplash | 무료 이미지 삽입 |
| Iconify | 다양한 아이콘 삽입 |
| Wireframe | 기본 와이어프레임 도형 제공 |
| Content Reel | 샘플 텍스트/이름/이미지 자동 삽입 |

Figma는 단순한 와이어프레임 도구를 넘어, **실제 개발팀과의 협업**에도 최적화된 툴인 것 같다.
