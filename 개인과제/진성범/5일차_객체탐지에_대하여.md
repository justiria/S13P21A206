# SSAFY 과제

## 📌 객체 탐지(Object Detection)

### 1. 개념 정의

객체 탐지(Object Detection)는 **디지털 이미지나 영상에서 사람이 관심 있는 객체를 식별하고, 해당 객체의 위치를 찾는 컴퓨터 비전 기술**이다. 단순히 이미지 안에 무엇이 있는지 분류하는 것을 넘어서, **어떤 객체가 어디에 있는지를 바운딩 박스(Bounding Box)와 함께 출력**한다.

- 예시: 이미지에 3명의 사람이 있다면 각각 사람(PERSON)이라는 라벨과 좌표 정보가 함께 출력됨.
- 대표적인 출력 형태: `클래스명 + 좌표(x, y, w, h)`

### 2. 기술적 원리

객체 탐지 알고리즘은 일반적으로 다음의 과정을 거친다:

### (1) 특징 추출(Feature Extraction)

- CNN(합성곱 신경망, Convolutional Neural Network)을 이용해 이미지의 저수준 특징(모서리, 패턴 등)을 추출

### (2) 후보 영역 생성(Region Proposal)

- 이미지 내 객체가 있을 가능성이 높은 영역을 선택
    - 예: Selective Search, Anchor box, Sliding window 등

### (3) 클래스 분류 + 위치 예측

- 각 후보 영역에 대해 어떤 객체인지 분류(Classification)
- 동시에 해당 객체의 위치(좌표)를 예측(Regression)

### (4) 비최대 억제(NMS, Non-Maximum Suppression)

- 중복된 탐지 결과 중 신뢰도(Confidence Score)가 높은 하나만 남김

### 3. 대표 알고리즘

| 알고리즘 | 특징 |
| --- | --- |
| **Faster R-CNN** | 정확도 높음, 다소 느림 (2단계: Region Proposal + Detection) |
| **YOLO (You Only Look Once)** | 한 번에 예측 → 빠르고 실시간 가능 |
| **SSD (Single Shot Multibox Detector)** | 속도와 정확도의 균형 우수 |

---

### 4. 장점과 단점

### ✅ 장점

- **위치 정보까지 제공**: 객체가 어디 있는지 정확한 위치 좌표 제공
- **다중 객체 탐지**: 이미지/영상 내 다수의 객체 동시 탐지 가능
- **실시간 처리 가능**: YOLO, SSD 등의 경량화 모델로 빠른 속도 구현
- **다양한 분야 확장성**: 의료, 보안, 교통 등 응용 가능 분야 넓음

### ❌ 단점

- **학습 데이터 요구량 큼**: 수천~수만 장의 바운딩 박스 주석 필요
- **복잡한 환경에서 정확도 저하**: 조명, 배경, 가려짐 등으로 탐지 성능 감소
- **소형 객체 탐지 어려움**: 객체가 작거나 겹쳐 있으면 탐지 성능 저하
- **고사양 연산 자원 필요**: GPU 등 고성능 하드웨어 요구

### 5. 주요 활용 분야

| 분야 | 구체적 활용 예시 |
| --- | --- |
| **자율주행 자동차** | 차량, 보행자, 신호등 등의 탐지 및 판단에 사용 |
| **보안 및 감시 시스템** | CCTV 영상에서 침입자, 이상 행동 자동 탐지 |
| **의료 영상 분석** | CT, MRI 영상에서 병변(종양, 출혈 등) 탐지 |
| **스마트 팩토리/로봇 비전** | 부품 불량 검사, 로봇 경로 탐색 및 작업 인식 |
| **스마트 리테일** | 매장 내 고객 동선 분석, 상품 탐지, 무인 결제 |
| **모바일 앱, AR/VR** | 얼굴 인식, 오브젝트 인식 기반 기능 구현 |
| **농업/환경 분야** | 병해충 탐지, 작물 식별, 폐기물 모니터링 |

---

### 6. 마무리

객체 탐지는 인간의 시각적 인지를 인공지능이 모방하는 대표적인 기술로, 다양한 산업 분야에서 핵심적인 역할을 하고 있다. 특히 실시간 인식, 자동화 기술이 중요해지는 미래 사회에서 객체 탐지 기술의 중요성은 더욱 커질 것으로 전망된다.
