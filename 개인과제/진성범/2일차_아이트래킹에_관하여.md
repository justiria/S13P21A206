## 📌 1. 아이트래킹(Eye Tracking)의 개념

**아이트래킹**은 사람의 **눈동자 움직임(시선, 주시점, 동공 크기 등)**을 **실시간으로 추적 및 기록**하는 기술입니다.

이를 통해 **어디를, 얼마나 오래, 어떤 순서로 보는지** 등의 시각적 주의 정보를 분석할 수 있습니다.

### 작동 원리

- 적외선(IR) 센서나 고속 카메라를 통해 **눈동자의 움직임**을 감지하고, 이를 기반으로 **시선 방향과 초점 위치**를 계산.
- 시선의 **고정(fixation)**, **도약(saccade)**, **눈 깜빡임(blink)** 등을 분석.

---

## ✅ 2. 장점

| 장점 | 설명 |
| --- | --- |
| **비언어적 반응 측정 가능** | 사용자의 무의식적 주의나 관심을 객관적으로 파악 가능 |
| **실시간 데이터 수집** | 시선 움직임을 실시간으로 기록, 빠른 피드백 가능 |
| **사용자 행동 분석에 효과적** | 제품 설계, 웹사이트 UI/UX 개선 등에 활용 가능 |
| **생체 신호 기반 정확도** | 다른 설문 등보다 조작 가능성이 낮음 |

---

## ⚠️ 3. 단점

| 단점 | 설명 |
| --- | --- |
| **고가 장비 필요** | 고정밀 장비(안구추적기)는 가격이 높음 |
| **개인차 영향** | 안경, 렌즈, 눈의 구조, 조명 조건 등에 따라 정확도 차이 발생 |
| **해석 난이도** | 시선만으로 ‘이유’를 파악하기는 어려워, 추가적 해석 도구 필요 |
| **피로 유발 가능성** | 장시간 추적 시 사용자 피로 발생 가능 |

---

## 🔧 4. 쓰임새 (활용 분야)

| 분야 | 활용 사례 |
| --- | --- |
| **UX/UI 디자인** | 웹사이트, 앱, 광고에서 사용자 시선 흐름 분석 → 정보 배치 최적화 |
| **마케팅/광고** | 제품 포장, 광고물 시선 집중도 확인 → 효과적인 노출 전략 수립 |
| **심리학/인지과학** | 주의력, 학습능력, 감정반응 등 연구 |
| **자동차/항공** | 운전자/파일럿의 집중도 분석 및 피로 모니터링 시스템 |
| **의료/재활** | 뇌 손상 환자의 시각 주의 훈련, 눈으로 조작하는 보조기기 개발 |
| **게임/VR/AR** | 몰입도 분석, 사용자 인터랙션 최적화 |
| **교육** | 학습자 시선 분석을 통한 학습 자료 개선 및 주의력 측정 |
